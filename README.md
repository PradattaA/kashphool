Follow these instructions to run the application locally for development.

### Prerequisites

You need Node.js and a package manager (npm or yarn) installed on your machine.

-   [Node.js (LTS recommended)](https://nodejs.org/ "null")


### Setup and Run

1.  **Clone the repository:**

    ```
    git clone [https://github.com/PradattaA/kashphool.git](https://github.com/PradattaA/kashphool.git)
    cd kashphool
    ```

2.  **Install dependencies:**

    ```
    npm install
    # or
    yarn install
    ```

3.  **Start the development server:**

    ```
    npm start
    # or
    yarn start

    ```
The application will now be running on `http://localhost:3000`.
